What can I hide in an image?

Currently, either text or another file can be hidden in an image.

What image formats are supported?

All image formats supported by Android can be used. Currently these are JPEG, GIF, PNG, BMP, WebP and HEIF. For technical reasons the image is always saved as PNG, because this image format allows a lossless compression of the pixels. This is the only way to ensure that the hidden data can be extracted again.

What files can I hide in an image?

Any type of file can be hidden in an image as long as it is large enough.

How can I hide data encrypted?

When hiding data, it is selectable whether the data should be hidden encrypted or not. Encryption requires a password, which must also be entered when extracting. Without the password it is not possible to extract the hidden data.

What is the maximum size of an image?

For technical reasons, only images up to 25 MB in size can be processed in the app.

Â 

How much data can I hide in an image?

To hide data 4 bytes of image information are needed per byte. This means that an image can contain a maximum of 25% hidden data.
