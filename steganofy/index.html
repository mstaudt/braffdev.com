Steganofy

Wolltest du schon immer mal eine geheime Nachricht in einem Foto verstecken? Mit Steganofy ist das nun möglich!

 

Verstecken & Extrahieren

  • Verstecken von Nachrichten oder Dateien in Bildern
  • Extrahieren von versteckten Daten aus Bildern

 

Sicherheit

  • Daten können mit AES-256 verschlüsselt werden
  • Der Code der App ist öffentlich und kann jederzeit eingesehen werden

 

Datenschutz

  • Kein Tracking
  • Alle Daten bleiben auf deinem Gerät

 

Unterstützte Bild- und Dateiformate

  • Folgende Bildformate werden unterstützt: JPEG, GIF, PNG, BMP, WebP, HEIF
  • Alle Arten von Dateien können in einem Bild versteckt werden, sofern das Bild groß genug ist

 

Wichtige Hinweise zum Versenden von Bildern

  • Messaging-Apps, die Bilder komprimieren (z.B. Facebook oder WhatsApp), werden die versteckten Daten nicht enthalten

Steganofy - Häufig gestellte Fragen

Was kann ich in einem Bild verstecken? Aktuell kann in einem Bild entweder Text oder eine andere Datei versteckt werden.  Welche Bildformate werden unterstützt? Alle von Android unterstützen Bildformate können genutzt werden. Aktuell sind das JPEG, GIF, PNG, BMP, WebP und HEIF. Aus technischen Gründen wird das Bild immer als PNG gespeichert, da dieses Bildformat eine verlustfreie Kompression der Pixel erlaubt. Nur so ist gewährleistet, dass die versteckten Daten auch wieder extrahiert werden können....
